Contact Type Management

Developed by Yantradhigam Labs Private Limited

🧩 Overview
The Contact Type Management module efficiently organizes and categorizes contacts with customizable contact types including vendors, customers, and employees. This module enhances your partner classification and improved relationship management.
This module allows you to:

✅ Create and manage contact types (Customer, Vendor, Distributor, etc.)
✅ Assign multiple contact types to partners
✅ Filter and search partners by contact type
✅ Generate automatic contact codes
✅ Enhanced partner management workflow

🚀 Features

Automatic sequence generation for contact codes
Many-to-many relationship between partners and contact types
Enhanced search and filter capabilities
Tree, form, and kanban views
Integration with existing partner workflow
Customizable contact type categories

📋 Contact Type Categories
The module supports various contact type categories:

Customer - For client relationships
Vendor - For supplier management
Employee - For internal staff
Distributor - For distribution network
Supplier - For procurement management
Partner - For business partnerships
Contractor - For external service providers

🔧 Installation & Configuration

Install the module from Odoo Apps
Navigate to Contacts > Configuration > Contact Type
Create contact types that match your business needs
Assign contact types to partners during creation or editing
Use filters to organize and search partners effectively

📊 Usage
Creating Contact Types

Go to Contacts > Configuration > Contact Type
Click Create to add a new contact type
Enter the contact type name and description
Save the contact type

Assigning Contact Types to Partners

Open any partner record
Navigate to the Contact Types field
Select one or multiple contact types
Save the partner record

Filtering Partners by Contact Type

Go to Contacts
Use the Filters menu
Select Contact Type filter
Choose the desired contact type(s)
