How to Use Contact Type Management

Simple steps to implement and start organizing your contacts with smart type classification.

1
Install Module

Install the Contact Type Management app from Odoo App Store and access configuration settings.

2
Define Contact Types

Create your contact types (Customer, Vendor, Employee, Distributor) to match your business needs.

3
Assign Types

Assign single or multiple contact types to partners during creation or editing process for better organization.

4
Filter & Manage

Use enhanced search and filter options to quickly find and manage partners based on their contact types.
