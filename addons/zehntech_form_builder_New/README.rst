Odoo Form Builder
================================================================

The **Odoo Form Builder** is a powerful no-code module that enables users to create, customize, and publish professional online forms and surveys directly within Odoo. With its intuitive drag-and-drop interface, users can design flexible, data-driven forms that seamlessly integrate with Odoo’s backend.  

This **Odoo Form Builder Odoo App** allows users to create, customize, and publish professional online forms and surveys using a no-code, drag-and-drop interface. It supports all necessary field types (Text, Email, Dropdown, File Upload, Captcha, Rating, etc.) with full control over validation and labels. Forms are instantly shareable via public links, QR codes, or embed codes without requiring an Odoo login. The module includes a dashboard for managing submissions, exporting data (CSV), and tracking performance with regional analytics. Essential features like email notifications and real-time styling ensure a smooth, integrated data collection experience within Odoo.

**Key Features**
================================================================

- **Drag & Drop Field Reordering**  
  Effortlessly organize and reorder form fields with a simple drag-and-drop interface for quick customization.  

- **Multiple Field Types**  
  Build powerful and versatile forms with a wide range of field types, including Text, Email, Dropdown, File Upload, Rating, Captcha, and more.  

- **Real-Time Customization & Live Preview**  
  Instantly personalize your form's appearance with color, font, and layout adjustments while previewing every change in real time.  

- **Flexible Form Availability Control**  
  Schedule forms to be active always, between specific dates, or within defined time ranges.  

- **Public Sharing via URL or QR Code**  
  Instantly generate public shareable links and QR codes that allow anyone to access and submit the form without login.  

- **Embed Forms on Any Website**  
  Seamlessly embed your forms into any external website while retaining full responsiveness and styling.  

- **Email Alerts & Response Analytics**  
  Send automated notifications to form owners and respondents while tracking submissions, conversion rates, and regional insights through a built-in analytics dashboard.  

- **Response Management & Export**  
  View, filter, and export all form submissions to CSV for easy offline reporting.  

- **Multi-Language Support**  
  Create and publish forms in multiple languages including English, Japanese, French, German, and Spanish.  

- **Publish/Unpublish Control**  
  Easily enable or disable forms with personalized messages for expired or unpublished states.

**Installation**
================================================================

1. Copy the module folder to your Odoo custom addons directory.  
2. Update the Apps List in Odoo.  
3. Search for **Odoo Form Builder** and install it.  
4. Navigate to **Form Builder** in the main Odoo menu.  
5. Start creating forms, adding fields, configuring layouts, and enabling notifications.  

**How to use this module**
================================================================

1. Go to **Form Builder > Create** to design a new form.  
2. Add fields using the add a line interface and configure labels, types, placeholders, and validation rules.  
3. Choose between **Default (Vertical)** or **Horizontal** layouts for form design.  
4. Enable **Email Notifications** for owners or respondents as needed.  
5. Set **Form Availability** and **Response Limits** to control access.  
6. Share forms via **Public URL**, **QR Code**, or **Embed Code**.  
7. View submissions in the **Responses** tab and analyze performance in the **Analytics Dashboard**.  
8. Use **Customization Panel** to adjust colors, fonts, and layout, and publish your final design.
